# Cookbook

A collection of recipes for common use cases and patterns with Castor Ledgering.

## Recipes

### Currency and Exchange

- [Currency Exchange](currency-exchange.md) - Exchange between different currencies using linked transfers

### Balance Management

- [Balance Conditional Transfers](balance-conditional-transfers.md) - Enforce balance constraints using account flags
- [Balance Invariant Transfers](balance-invariant-transfers.md) - Automatically calculate transfer amounts to balance accounts

### Transfer Workflows

- [Pending Transfers](pending-transfers.md) - Two-phase transfer workflows with reservations

## Recipe Format

Each recipe follows this structure:

1. **Problem**: What problem does this solve?
2. **Solution**: How to implement it
3. **Example**: Working code example
4. **Considerations**: Important notes and edge cases
